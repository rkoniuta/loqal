## Tech Stack

- React
- TypeScript
- Tailwind CSS
- React Router

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/advert-app.git
cd advert-app
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```

The application will be available at `http://localhost:3000`
